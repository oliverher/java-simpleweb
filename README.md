webapp-simpleweb
========

#### What is this?

It's a simple Maven Archetype to be used as a template to Java WebProjects, the following frameworks are already
configured with basic options:

* [JSF 2.2]
* [Hibernate 5]
* [Spring MVC 4.2.0]
* [Spring Security 4.0.2]
* [AspectJ 1.8.6]
* [slf4j 1.17.12]
* [logback 1.1.3]
* [HikariCP 2.4.1]
* [Postgresql Connector]
